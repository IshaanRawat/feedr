About Feedr

Feedr is a feed aggregating web app, which will allow users to read all their favorite top blog's posts all at one place.

We built it clean, simple and mobile-friendly for the users to use and understand. With the help of a News API Provider and with the most powerful web framework, React.js, we built this app to provide users the top posts of their interest.

Resources Used

  • React.js
  • NewsAPI
  • Atom
  • Adobe Illustrator
